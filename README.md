# Amazon Clone

A front-end clone of the Amazon homepage built using **HTML** and **CSS**.

## 🔥 Features

- Amazon-style header with logo, search bar, and cart icon  
- Product categories and sections  
- Responsive layout (basic)  
- Footer with links

## 🛠️ Technologies Used

- HTML5  
- CSS3

## 📁 Folder Structure

Amazon-clone/
├── index.html
├── style.css
├── images/
